# Java OOP: 20 Semester Questions

This small collection contains 20 Java OOP example programs (one per file). Each file demonstrates a common OOP topic useful for a college semester.

Files:
1. Q01_Encapsulation.java — Encapsulation with Student
2. Q02_Inheritance.java — Simple inheritance (Animal/Dog)
3. Q03_PolymorphismCompileTime.java — Method overloading example
4. Q04_PolymorphismRuntime.java — Method overriding / runtime polymorphism
5. Q05_Abstraction.java — Abstract class Shape
6. Q06_Interface.java — Interface implementation (Movable)
7. Q07_AbstractClass.java — Abstract class with concrete methods
8. Q08_ConstructorOverloading.java — Constructors overloading (Box)
9. Q09_MethodOverloading.java — Calculator overloading
10. Q10_MethodOverriding.java — Method overriding demo
11. Q11_Association.java — Association (Library & Book)
12. Q12_Aggregation.java — Aggregation (Department & Student)
13. Q13_Composition.java — Composition (House & Room)
14. Q14_StaticMembers.java — Static fields and methods
15. Q15_NestedClasses.java — Inner and static nested classes
16. Q16_AnonymousClasses.java — Anonymous class example
17. Q17_ExceptionHandlingOOP.java — Custom exception used in OOP context
18. Q18_Singleton.java — Singleton pattern
19. Q19_ObserverPatternSimple.java — Simple observer pattern
20. Q20_FactoryPattern.java — Simple factory pattern

How to compile:

```bash
javac *.java
```

How to run an example (replace class name):

```bash
java Q01_Encapsulation
```

Notes:
- Code aims to be minimal and readable, with few comments as requested.
- No packages used so you can compile in one directory.
